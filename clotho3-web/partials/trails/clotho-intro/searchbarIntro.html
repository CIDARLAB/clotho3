Example Commands

You can declare variables in Javascript syntax: var foo = 'bar'

You can run complex queries over all objects in your database, using clotho.query(myParams), where myParams is an object with key-value filters

Basic DNA manipulation functions can be directly accessed, for example: DNA.complement('acgttgca')

For this demo, we'll be sticking to the easier syntax.

Other functions can be run by calling clotho.run.
