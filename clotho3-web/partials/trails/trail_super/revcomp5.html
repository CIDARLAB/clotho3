Bootstrap Another Angular App

This approach does not rely entirely on mixing in components - components may be added to the primary app stack, but here the widget is its own angular app.

This approach calls Application.script(), a cache-busting function to execute a script every time by time-stamping it. The script bootstraps the widget and inserts it into the div below. This template has no associated controller
