Using the Command Bar

The Command Bar provides a simple interface for running functions in Clotho.

Issuing commands

Type f to focus the Command Bar input.

Words are tokenized as you input them. When they match something in the database, you'll see a list of autocompletions. These include functions, data, and schemas.

Hover over an autocompletion (or use your up and down arrows) to see more information about it.

Activity Log

The activity log tracks your commands and their responses with Clotho. Responses appear briefly in a snippet to the right of the input, but exist throughout your session hidden away.

Type a to show the Activity Log.
