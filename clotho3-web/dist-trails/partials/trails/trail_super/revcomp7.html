jQuery Mixin

It is likely that many developers will wish to stick to jQuery. Several jQuery extensions exist. jQuery is a language for manipulating the DOM, and as such it will likely reference the template being added to the page.

script and dependencies run before the template / video / etc. is loaded and compiled. A parameter onload may be used in the case where something must be run after the template has been added to the DOM - as is the case here. The typeahead plug-in is loaded via dependencies, and the jQuery binding it to the template is loaded via onload. onload is cache-busting (it simply calls Application.script after the template has been loaded).
