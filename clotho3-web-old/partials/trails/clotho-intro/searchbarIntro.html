Example Commands

You can declare variables in Javascript syntax: var foo = 'bar'

You can retrieve objects by name from the Clotho database: clotho.get('GFPmut3')

You can run complex queries over all objects in your database, using clotho.query(myParams), where myParams is an object with key-value filters: clotho.query({"name":"GFPmut3"})

DNA manipulation functions can be directly accessed, for example: DNA.complement('acgttgca')
