The ngSanitize module provides functionality to sanitize HTML.

Installation

As a separate module, it must be loaded after Angular core is loaded; otherwise, an 'Uncaught Error: No module: ngSanitize' runtime error will occur.

  <script src="angular.js"></script>
  <script src="angular-sanitize.js"></script>

Usage

To make sure the module is available to your application, declare it as a dependency of you application module.

  angular.module('app', ['ngSanitize']);
